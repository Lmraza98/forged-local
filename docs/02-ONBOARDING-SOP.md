# ForgedLocal Onboarding SOP

## Overview
30-minute checklist to get a new client live. Promise: "Up and running in 48 hours."

---

## Pre-Onboarding (Before Call)

### Internal Prep Checklist
- [ ] Review lead notes from CRM
- [ ] Look up their Google Business listing
- [ ] Note their current phone number
- [ ] Check phone carrier/type if known
- [ ] Prepare tracking number to provision

**Default Message Template:**
```

- Any specific services to mention?

```

**After-Hours Handling:**
```

```

---

### Part 3: Technical Setup (5 min)

**Call Forwarding Instructions by Carrier:**

**Verizon Wireless:**
```
1. Dial *72
2. Enter ForgedLocal tracking number
3. Press Send/Call
4. Wait for confirmation tone
To disable: Dial *73
```

**AT&T Wireless:**
```
1. Dial *21*[ForgedLocal number]#
2. Press Send/Call
To disable: Dial #21#
```

**T-Mobile:**
```
1. Dial **21*[ForgedLocal number]#
2. Press Send/Call
To disable: Dial ##21#
```

**Landline/VoIP:**
```
Usually: *72 + number
Or configure in admin portal
```

**Test the Setup:**
- [ ] Call their business number from test phone
- [ ] Verify call routes through ForgedLocal
- [ ] Let it ring to voicemail
- [ ] Confirm auto-text is sent within 60 seconds
- [ ] Reply to text, confirm it appears in inbox

- [ ] How's the system working?
- [ ] Any missed texts or issues?
- [ ] Review calls/leads so far
- [ ] Adjust auto-text if needed

### End of Pilot (Day 14):
- [ ] Schedule review call
- [ ] Prepare metrics summary

---

## Troubleshooting Common Issues

### "Texts not going out"
1. Verify call forwarding is still active
2. Check tracking number is provisioned
3. Test with another call

### "Getting too many spam texts"
1. Adjust after-hours settings
2. Add blocked number list

### "Want to change the message"
1. Note new message in CRM
2. Update in platform
3. Send test to confirm

### "Want to pause temporarily"
1. Remove call forwarding: *73 or ##21#
2. Note pause in CRM
3. Schedule reactivation reminder

---

## Definition of "Live"

A client is considered **LIVE** when:
1. ✅ Call forwarding is active
2. ✅ Test missed call triggers auto-text within 60 seconds
3. ✅ Client can see leads in inbox
4. ✅ Client has responded to at least one test lead
5. ✅ Emergency keywords are configured (if requested)

**Time to live target:** <48 hours from first call
